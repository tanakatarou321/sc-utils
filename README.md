# sc-utils
This is a utility that can be used in the SystemC code.

# Download
You can access the sc-utils Git repository from [here](https://github.com/tanakatarou321/sc-utils).

# Usage
## Setup
Add include directory
```
sc-utils/include
```
## build
Include below
```
#include "sc-utils.h"
```

# Functions
## print_hierarch()
Output hierachical structure
```
print_hierarch()
```
